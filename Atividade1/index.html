Introdução

Editores de código são provavelmente as ferramentas mais importantes para desenvolvedores web. Encontrar uma ferramenta que melhor se adeque ao workflow e aos objetivos, e que tenha todas as características necessárias pode resultar em uma enorme economia de tempo e melhorar significativamente a qualidade do seu código. Neste trabalho, serão mostrados os melhores editores de código para desenvolvedores web. Todos possuem versões para os três maiores sistemas operacionais, Windows, Linux e OS X.[1]
